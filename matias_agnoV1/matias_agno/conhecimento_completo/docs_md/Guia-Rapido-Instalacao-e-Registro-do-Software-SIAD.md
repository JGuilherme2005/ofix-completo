# Guia-Rapido-Instalacao-e-Registro-do-Software-SIAD

**Guia Rápido Instalação e Registro do Software SIAD – NA9000E:**


_**1.**_ **Instalando software de controle:**


Para realizar o instalação do software de controle SIAD, existe dois caminhos:


    - O CD fornecido com o equipamento (não recomendável).


    - Utilizar o link para download da base de conhecimento.


O link para download da base de conhecimento é o meio mais eficiente e seguro para a instalação do software do SIAD, pois
ele contém a versão mais recente do software.


Entre no endereço da internet:

# https://www.despoluir.napro.com.br


Neste site o usuário poderá encontrar informações, serviços e downloads do sistema SIAD.


Na base de conhecimento entre no artigo “Link para baixar o programa executável”. Como demonstrado na figura baixo:


Clique no link e siga as intruções de download.


Após o fim do download execute o arquivo de instalação. Normalmente o nome do arquivo será:

## setup_despoluir_4_29.exe


Onde a versão do software está identificada no nome do arquivo.


A janela do instalador aparecerá. Siga as instruções na tela. Dependendo do sistema instalado poderá aparecer algum aviso
de erro, como por exemplo


:


Clique em OK ou confirme para continuar a instalação.


Juntamente com o software de controle SIAD, outros aplicativos serão adicionados, sempre confirme as operações para a
instalação dos aplicativos.


Somente reinicialize o computador ao final da instalação do software de controle SIAD – NA9000E.


O software de controle possui um sistema de atualização automático pela internet, pode ocorrer de aplicativos antivírus ou
similares bloquearem a comunicação com a internet, favor liberar no seu software antivírus.

Na primeira instalação o usuário deverá registar o software. O sistema de registro não permite que o usuário utilize o sitema
até realizar o registro do software.


Para liberar o software mande para a Napro os dados
constantes da janela de Registro.


Forneça os dados do ID de Hardware e o nome ou identificação
da federação/sindicato, ou outro indicativo do usuário.


[Envie estes dados através do site www.despoluir.napro.com.br.](http://www.despoluir.napro.com.br/)


Na tela inicial do site, entre na opção “Enviar Ticket”.


Escolha a opção “Registro do Software”. Preencha todos os dados. Na opção Motivo do Contato, selecione dúvida técnica.


Em Assunto, informe Registro do Software. Coloque os dados do ID de hardware e identificação na Mensagem, você pode
também enviar um Print da tela acima.


Aguarde a resposta do suporte da Napro. Ao receber os dados insira no campo correspondente e clique em Registrar.


Esta operação só será necessária apenas uma vez.


Após o registro o usuário deverá realizar o primeiro cadastro no sistema.


Como não existe nenhum cadastro no sistema o usuário deverá na tela de Login utilizar os seguintes dados:


Usuário: **ADMIN**


Senha: **Admin**


Este usuário e senha só é válido quando não existe nenhum registro no sistema, após o registro do primeiro técnico não está
senha será descontinuada.


**É importante após cadastrar o Gerenciador e Operador (obrigatório), cadastrar o Administrador do**
**Sistema.**


Após o cadastro do gerenciador e operador, a opção de cadastrar os técnicos será liberada.


**O primeiro cadastro do Técnico deverá ser obrigatoriamente o Administrador do Sistema.**


Caso não seja cadastrado um Administrador o usuário poderá perder algumas funções do sistema, e deverá

desinstalar e realizar uma nova instalação.
